# NudgeHQ Team Onboarding Files

Share these with new NudgeHQ team members:

- `NudgeHQ_Team_Onboarding_Pack.pdf` — clean shareable PDF.
- `NudgeHQ_Team_Onboarding_Pack.docx` — editable Word/Google Docs version.
- `NudgeHQ_Team_Onboarding_Pack.md` — source Markdown for future updates.

The pack covers:

- Product problem and solution
- Target users and role logic
- Current MVP scope
- Product modules
- Architecture and APIs
- Pricing direction
- Implementation plan
- Future roadmap
- Team operating rules
- First-week onboarding checklist

Note: update this pack after major changes to product scope, pricing, backend routes, or launch plan.
